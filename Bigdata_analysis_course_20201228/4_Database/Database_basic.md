#### SQLite 데이터유형

| 정수    | 실수 | 텍스트                                                       | 시간     |
| ------- | ---- | ------------------------------------------------------------ | -------- |
| TINYINT | REAL | CHAR: 고정사이즈, 남는공간은 빈칸으로 채움                   | DATE     |
| INTEGER |      | VARCHAR: 가변적으로 공간의 양에따라 저장, 많이 사용하면 속도저하가 발생할 수 있음 | TIME     |
| BIGINT  |      | TEXT                                                         | DATETIME |
